MedAssist Mobile 医院陪诊 App 可视化流程图

目标：让老人每一刻只看到“现在要做什么”，并通过扫描、地图导航、材料提醒和休息求助完成就诊。

医院室内地图导航

Flow 1：开始就诊与识别目的

用户输入就诊目的或上传纸张，系统判断第一步应该去哪里。

触发入口：
▯ 首页：今天为什么来医院？
输入文字例如：化验、复诊、取药、挂号
点击
查找我的第一步
上传纸张挂号单、短信截图、处方、检查结果
模拟扫描识别
系统判断 取药 → 扫描处方
复诊 / 结果 → 扫描检查结果
化验 / 检查 → 扫描医嘱单
其他 → 扫描已有纸张
进入对应的下一步页面

Flow 2：扫描纸张并生成下一步指示

用户点击扫描，上传任意图片后，页面显示模拟识别结果、识别线索和下一步操作。

扫描触发：
📷 扫描纸张 / 医嘱 / 处方 / 检查结果
点击
立即扫描
选择图片
识别结果卡片 显示文件名
显示识别摘要
显示科室 / 楼层 / 材料 / 窗口线索
确认目的地 → 挂号
规划化验路线 → 缴费
返回医生处 → 复诊
去药房 → 取药

Flow 3：医院室内地图导航

导航页面模拟百度地图式室内导航：顶部目的地、楼层切换、蓝色路线、当前位置、目的地和下一步方向。

导航触发：
🗺 需要去挂号、分诊、诊室、缴费机、化验室或药房
地图页面
你 终 化验室 电梯 药房
底部方向卡 直走 120 米
到电梯处右转
点击 Repeat 可重复路线
到达后点击
我已到达

Flow 4：挂号到看医生的完整路径

从确认目的地开始，依次完成挂号、准备材料、分诊、排队和进入诊室。

导航到挂号窗口
我已到达
准备挂号材料 点选身份证
点选医保卡 / 就诊卡
点选预约码
听工作人员讲话，保存窗口提示
扫描新挂号单，识别分诊位置
导航到分诊台，操作分诊机
记录排队号，等待叫号
导航到诊室并记录医生讲话

Flow 5：检查、复诊与取药路径

医生开检查后，系统引导缴费、化验、回诊、扫描处方并最终到药房取药。

扫描医生医嘱单
识别到：先缴费，再去 2F 化验室
地图导航到自助缴费机 B
操作缴费，保存收据
地图导航到 2F 化验室
提交材料并完成采样
扫描检查结果，返回医生处
记录复诊建议，扫描处方
导航到药房并取药

Flow 6：全局辅助功能

所有页面都保留休息、返回、快捷选项和后台事件记录，保证老人可以随时暂停或求助。

✓ 任意页面
点击 Rest
打开休息弹层，点击 Resume Visit 返回原页面
点击 Back
返回上一个医院步骤
点击主按钮
记录 step_completed，并进入下一步
点击 family / staff
记录家属更新或工作人员求助事件
